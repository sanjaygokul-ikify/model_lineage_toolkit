# Contributing to Model Lineage Toolkit

Thank you for considering contributing to Model Lineage Toolkit! We welcome contributions from anyone interested in improving the library.

## PR Workflow

1. Fork the repository using `git fork https://github.com/your-username/model_lineage_toolkit.git`
2. Create a new branch using `git branch feature/your-feature`
3. Make your changes and commit them using `git commit -m "Your commit message"`
4. Open a pull request using `git push origin feature/your-feature`

## Commit Standards

* Use meaningful commit messages
* Follow the standard commit format: `type(scope): brief description`

## Code Style Rules

* Follow PEP 8 conventions
* Use consistent indentation and spacing
